Pour lancer l'interface graphique afin d'utiliser le chiffrement et le déchiffrement:
En ligne de commande, dans le bon répertoire, entrez :
- python Project.py


Pour obtenir le hash d'un fichier x avec un algorithme y:
En ligne de commande, dans le bon répertoire, entrez :

- python Hash.py "y" "x"

2 lignes apparaîtront,
la première correspond au hash sans sel
la deuxième correspond au hash avec sel
